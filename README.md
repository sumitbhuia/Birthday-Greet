
# Birthday Greet

This is my first android project , a android application completely written in kotlin . This is a very simple application that greets  birthday wish when someones's name is entered in the textbox.

## Lessons Learned

1. Handling .xml files
2. Handling .kt files
3. About widgets
4. About Widget constrains and different attributes
5. Intent passing between activities
